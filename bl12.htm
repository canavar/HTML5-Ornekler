History API Demo
popstate event'i ve history.state

pushState ile state ekleme :

history.pushState({foo: 'bar'},
   'Sayfa Başlığı',
   '/bar.htm');

popState ile state'i alma :

window.addEventListener("popstate", function(e) {
   var foo = history.state.foo;
}, false);
